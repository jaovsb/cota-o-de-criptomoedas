# Sobre o projeto

Projeto criado para identificar cotação de criptomoedas em Real(R$). 

# Tecnologias utilizadas

- React.js
- Coingecko api

# Como executar o projeto

```bash
# Clonar repositório
  git clone https://github.com/jaovsb/cotacao-de-criptomoedas.git

# Instalar dependências
  yarn install
  
# Iniciar projeto
  yarn start
```
